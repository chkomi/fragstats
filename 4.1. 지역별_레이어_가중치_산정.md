# 4.1. 지역별 레이어 가중치 산정 (개선된 방법론)

사용자님의 지적에 따라, 나주와 화순 두 지역의 특성이 다를 수 있다는 점을 고려하여 레이어별 가중치를 단순 평균하는 대신, **각 지역별로 레이어 가중치를 별도로 산정**하는 개선된 방법론을 제안하고 그 결과를 제시합니다.

이 접근법은 각 지역의 데이터 특성을 더욱 정밀하게 반영하여, 지역 맞춤형 정책을 수립하는 데 더 효과적인 근거를 제공합니다.

---

### 1. 분석 방법론

1.  **지역별 데이터 분리:** 이전 분석에서 도출된 8개 평가 대상의 최종 점수를 '나주' 그룹과 '화순' 그룹으로 분리합니다.
2.  **지역 내 정규화 (가중치 산정):** 각 그룹(지역) 내에서, 4개 레이어 점수의 총합을 구합니다. 그 후, 각 레이어 점수를 해당 지역의 총합으로 나누어 표준화(normalize)합니다. 이 값이 해당 지역 내에서의 '지역별 레이어 가중치'가 됩니다.
    *   `지역별 레이어 가중치 = (해당 레이어의 점수) / (해당 지역 4개 레이어 점수의 총합)`

### 2. 나주(Naju) 지역 레이어 가중치 산정

#### 2.1. 나주 지역 점수

| 평가 대상 ID | 종합 점수 |
| :--- | :---: |
| `naju_toyang` | 68.80 |
| `naju_nongeup` | 51.77 |
| `naju_infra` | 34.48 |
| `naju_pibok` | 23.90 |

#### 2.2. 나주 지역 가중치 계산

*   **나주 점수 총합:**
    `68.80 + 51.77 + 34.48 + 23.90 = 178.95`

*   **나주 최종 레이어별 가중치:**
    *   **W' naju_toyang** = 68.80 / 178.95 = **0.3845**
    *   **W' naju_nongeup** = 51.77 / 178.95 = **0.2893**
    *   **W' naju_infra** = 34.48 / 178.95 = **0.1927**
    *   **W' naju_pibok** = 23.90 / 178.95 = **0.1336**

### 4. 결론 및 활용 방안

지역별로 레이어 가중치를 산정한 결과, 두 지역 모두에서 **토양(Toyang)**의 중요성이 가장 높게 나타났으며, 그 뒤를 **농업용도(Nongeup)**가 잇는 것으로 확인되었습니다. 이는 수정된 토양 데이터가 분석 결과에 큰 영향을 미쳤으며, 두 지역 모두에서 토양의 질이 농업적 가치를 판단하는 핵심 요인임을 명확히 보여줍니다.

*   **나주**는 **토양(38.5%)**의 중요성이 가장 높고, 그 다음으로 농업용도(28.9%)가 중요합니다.
*   **화순** 역시 **토양(36.5%)**이 가장 중요하지만, 농업용도(27.9%)와의 격차는 나주보다 적어, 두 요인이 비교적 대등하게 가치에 기여합니다.

이처럼 지역별로 도출된 고유한 가중치 값을 GIS 중첩 분석에 적용하면, 각 지역의 특성을 반영한 정밀한 정책 수립이 가능해집니다.

*   **나주 최종 점수** = `(Toyang 지도 × 0.385) + (Nongeup 지도 × 0.289) + ...`
*   **화순 최종 점수** = `(Toyang 지도 × 0.365) + (Nongeup 지도 × 0.279) + ...`
