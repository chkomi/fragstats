제1장. 엔트로피 가중치법 개요
================================================================================

1-1. 엔트로피(Entropy)의 개념
--------------------------------------------------------------------------------

엔트로피는 정보이론에서 유래한 개념으로, 시스템의 무질서도 또는 불확실성을 측정하는
척도입니다. 데이터 분석에서는 "정보량"을 나타냅니다.

【핵심 원리】
- 엔트로피가 높다 = 데이터가 균일하게 분포 = 정보량이 적다 = 낮은 가중치
- 엔트로피가 낮다 = 데이터가 불균일하게 분포 = 정보량이 많다 = 높은 가중치

【예시】
지표 A: 모든 지역의 값이 50점으로 동일
  → 엔트로피 높음 → 변별력 없음 → 낮은 가중치

지표 B: 지역별로 10점~90점으로 큰 차이
  → 엔트로피 낮음 → 변별력 높음 → 높은 가중치


1-2. 다기준 의사결정(MCDM)에서의 활용
--------------------------------------------------------------------------------

농업진흥지역 평가는 다수의 지표(31개 Class 지표, 19개 Land 지표 등)를 종합해야 하는
다기준 의사결정 문제입니다.

【전통적 방법의 한계】
- 전문가 판단: 주관성 개입, 전문가마다 의견 상이
- 균등 가중치: 지표 간 중요도 차이 무시
- 계층분석법(AHP): 쌍대비교 과정의 복잡성, 일관성 문제

【엔트로피 가중치법의 장점】
✓ 객관성: 데이터 자체의 특성에서 가중치 도출
✓ 재현성: 동일한 데이터에서 항상 동일한 결과
✓ 효율성: 복잡한 설문이나 전문가 판단 불필요
✓ 투명성: 계산 과정이 명확하고 검증 가능


1-3. 농업진흥지역 평가에의 적용 의의
--------------------------------------------------------------------------------

【현행 제도의 문제점】
- 명확한 정량적 기준 부재
- 지역별 일관성 없는 판단
- 주관적 요소에 따른 형평성 문제

【엔트로피 가중치법 도입 효과】
✓ 과학적 근거: FRAGSTATS 경관지표 + 엔트로피 가중치
✓ 공정성: 데이터 기반 객관적 평가
✓ 정책 신뢰성: 투명한 의사결정 과정
✓ 지역 간 비교 가능: 표준화된 평가 체계


================================================================================
제2장. 엔트로피 가중치법 계산 과정
================================================================================

2-1. STEP 1: 데이터 수집 및 구축
--------------------------------------------------------------------------------

【데이터 구조】
- 평가 대상(m): 지역(화순, 나주) × 레이어(infra, toyang, nongeup, pibok)
- 평가 지표(n): Class 레벨 31개 지표
- 데이터 행렬 X: m×n 행렬

예시:
         | PLAND | LPI  | AI   | ... | CORE_MN |
---------|-------|------|------|-----|---------|
화순_infra | 4.14  | 0.26 | 89.6 | ... | 2.79    |
화순_toyang| 25.8  | 1.47 | 93.1 | ... | 36.2    |
나주_infra | 19.8  | 0.42 | 89.7 | ... | 3.84    |
...


2-2. STEP 2: 데이터 정규화
--------------------------------------------------------------------------------

【정규화의 필요성】
- 지표마다 단위와 범위가 다름 (%, ha, 개수 등)
- 비교 가능한 척도로 변환 필요

【Min-Max 정규화 공식】

(1) 긍정 지표 (높을수록 좋음: PLAND, LPI, AI 등)

    r_ij = (x_ij - min_j) / (max_j - min_j)

(2) 부정 지표 (낮을수록 좋음: NP, PD, ED 등)

    r_ij = (max_j - x_ij) / (max_j - min_j)

where:
- x_ij: i번째 대상의 j번째 지표 원시값
- r_ij: 정규화된 값 (0~1 범위)
- min_j, max_j: j번째 지표의 최소값, 최대값

【정규화 결과】
- 모든 값이 0~1 사이로 변환
- 1에 가까울수록 "좋음"을 의미


2-3. STEP 3: 엔트로피 계산
--------------------------------------------------------------------------------

【엔트로피 공식】

E_j = -k × Σ(p_ij × ln(p_ij))

where:
- E_j: j번째 지표의 엔트로피
- k = 1/ln(m): 정규화 상수 (m은 평가 대상 수)
- p_ij = r_ij / Σr_ij: i번째 대상이 j번째 지표에서 차지하는 비중
- ln: 자연로그

【계산 예시】
지표 LPI (최대패치비율):
- 화순_infra: 0.26 → 정규화: 0.00
- 화순_toyang: 1.47 → 정규화: 0.65
- 나주_infra: 0.42 → 정규화: 0.09
- 나주_toyang: 1.51 → 정규화: 0.68

비중 계산:
- p_1 = 0.00 / (0.00+0.65+0.09+0.68) = 0.00
- p_2 = 0.65 / 1.42 = 0.458
- p_3 = 0.09 / 1.42 = 0.063
- p_4 = 0.68 / 1.42 = 0.479

엔트로피:
- k = 1/ln(4) = 0.721
- E_LPI = -0.721 × [0×ln(0) + 0.458×ln(0.458) + 0.063×ln(0.063) + 0.479×ln(0.479)]
- E_LPI = 0.576 (낮은 엔트로피 → 변별력 높음)


2-4. STEP 4: 분산도(Diversity) 계산
--------------------------------------------------------------------------------

【분산도 공식】

D_j = 1 - E_j

where:
- D_j: j번째 지표의 분산도
- E_j: j번째 지표의 엔트로피

【의미】
- D_j가 클수록 해당 지표의 정보량이 많음
- 평가 대상들을 잘 구분할 수 있는 지표


2-5. STEP 5: 가중치 계산
--------------------------------------------------------------------------------

【가중치 공식】

w_j = D_j / ΣD_j = (1 - E_j) / Σ(1 - E_j)

where:
- w_j: j번째 지표의 가중치
- Σw_j = 1 (모든 가중치의 합은 1)

【나주/화순 분석 결과】
┌──────────────┬─────────┬──────────┬─────────┬─────────┐
│    지표명    │ 엔트로피 │  분산도  │ 가중치  │  순위   │
├──────────────┼─────────┼──────────┼─────────┼─────────┤
│ LPI         │  0.576  │  0.424   │ 0.1454  │    1    │
│ CORE_MN     │  0.711  │  0.289   │ 0.0992  │    2    │
│ AREA_MN     │  0.726  │  0.274   │ 0.0941  │    3    │
│ AREA_MD     │  0.765  │  0.235   │ 0.0806  │    4    │
│ GYRATE_MN   │  0.801  │  0.199   │ 0.0683  │    5    │
│ ...         │  ...    │  ...     │ ...     │   ...   │
└──────────────┴─────────┴──────────┴─────────┴─────────┘


2-6. STEP 6: 종합 점수 계산
--------------------------------------------------------------------------------

【종합 점수 공식】

S_i = Σ(w_j × r_ij) × 100

where:
- S_i: i번째 평가 대상의 종합 점수 (0~100점)
- w_j: j번째 지표의 가중치
- r_ij: 정규화된 값

【레이어 가중치 적용】

최종점수_i = 지표점수_i × 레이어가중치_i

레이어 가중치 (분산도 기반):
- 토지피복(pibok): 44.28%
- 농업용지(nongeup): 36.31%
- 기반시설(infra): 18.61%
- 토양(toyang): 0.79%

제5장. 정책 적용 방안
================================================================================

5-1. 단계별 실행 전략
--------------------------------------------------------------------------------

【1단계: 전수 조사 (0~6개월)】
- 모든 농업진흥지역 GIS 데이터 구축
- FRAGSTATS 분석 실시
- 엔트로피 가중치 모델 적용
- 전국 통일 기준으로 점수 산정

【2단계: 등급 분류 (6~12개월)】
- 5개 등급으로 전국 농업진흥지역 분류
- 등급별 면적 및 분포 현황 파악
- 지역별 특성 분석

【3단계: 정책 수립 (12~18개월)】
- 1~2등급: 절대/우선 보전 구역 법제화
- 3등급: 정비사업 우선순위 및 예산 배분
- 4~5등급: 해제 로드맵 수립

【4단계: 시범 사업 (18~36개월)】
- 3등급 지역 정비사업 시범 실시
- 사업 전후 점수 변화 측정
- ROI 분석 및 모델 검증

【5단계: 본격 시행 (36개월~)】
- 전국 확대 시행
- 3~5년 주기 재평가
- 가중치 업데이트

--------------------------------------------------------------------------------

【정책적 효과】

✓ 지속가능성: 장기적 농업기반 유지

부록. 엔트로피 계산 상세 예시
================================================================================

【예시: LPI 지표의 엔트로피 계산】

1. 원시 데이터
   화순_infra: 0.26
   화순_toyang: 1.47
   화순_nongeup: 1.27
   화순_pibok: 0.33
   나주_infra: 0.42
   나주_toyang: 1.51
   나주_nongeup: 8.96
   나주_pibok: 1.31

2. 정규화 (Min-Max, positive)
   min = 0.26, max = 8.96
   r1 = (0.26-0.26)/(8.96-0.26) = 0.000
   r2 = (1.47-0.26)/(8.96-0.26) = 0.139
   r3 = (1.27-0.26)/(8.96-0.26) = 0.116
   r4 = (0.33-0.26)/(8.96-0.26) = 0.008
   r5 = (0.42-0.26)/(8.96-0.26) = 0.018
   r6 = (1.51-0.26)/(8.96-0.26) = 0.144
   r7 = (8.96-0.26)/(8.96-0.26) = 1.000
   r8 = (1.31-0.26)/(8.96-0.26) = 0.121

3. 비중 계산
   합계 = 0.000+0.139+0.116+0.008+0.018+0.144+1.000+0.121 = 1.546
   p1 = 0.000/1.546 = 0.000
   p2 = 0.139/1.546 = 0.090
   p3 = 0.116/1.546 = 0.075
   p4 = 0.008/1.546 = 0.005
   p5 = 0.018/1.546 = 0.012
   p6 = 0.144/1.546 = 0.093
   p7 = 1.000/1.546 = 0.647
   p8 = 0.121/1.546 = 0.078

4. 엔트로피 계산
   k = 1/ln(8) = 1/2.079 = 0.481
   E = -k × Σ(p×ln(p))
     = -0.481 × [0.090×ln(0.090) + 0.075×ln(0.075) + 0.005×ln(0.005)
                 + 0.012×ln(0.012) + 0.093×ln(0.093) + 0.647×ln(0.647)
                 + 0.078×ln(0.078)]
     = -0.481 × (-1.198)
     = 0.576

5. 분산도 및 가중치
   D = 1 - 0.576 = 0.424
   w = 0.424 / Σ(모든 지표의 D) = 0.1454

∴ LPI의 최종 가중치 = 14.54%
