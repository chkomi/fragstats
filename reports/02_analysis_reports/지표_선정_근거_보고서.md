# FRAGSTATS 58개 지표 선정 근거 및 상세 해설

## 1. 보고서 개요

본 보고서는 FRAGSTATS를 통해 산출된 58개 경관 지표가 농업적 가치 평가 모델에서 **'긍정(Positive)' 또는 '부정(Negative)'의 방향성을 갖는 이유**에 대한 학술적, 이론적 근거를 상세히 제시합니다. 각 지표의 정의, 측정 단위, 그리고 농업적 가치(생산성, 효율성, 지속가능성)와의 연관성을 `paper` 폴더 내 참고 문헌에 기반하여 기술합니다.

**[주요 인용 자료 약어]**

*   **FRAGSTATS Manual:** `FRAGSTATs Spatial Pattern Analysis Program for Quantifying Lanscape Structure.pdf`
*   **FRAGSTATS Intro:** `FRAGSTATS, A Free Tool for Quantifying and Evaluating Spatial Patterns.pdf`
*   **Trends in Metrics:** `Trends in the use of landscape spatial metrics as landscape indicators.pdf`
*   **LESA Handbook:** `National Agricultural Land Evaluation and Site Assessment (LESA) Handbook.pdf`
*   **우량농지 보전 정책:** `우량농지 보전을 위한 정책프로그램 개발.pdf`
*   **농지 보전 적지 분석:** `농업진흥지역 밖 농지의 보전 적지 분석 환경성과 생산성 중심으로.pdf`

## 2. 면적 및 밀도 (Area & Density Metrics)

농지의 면적, 비율, 밀도 관련 지표는 농업 생산의 가장 기본적인 요소인 '규모'를 측정합니다. 일반적으로 농지는 **규모가 크고, 파편화가 적을수록** 농업적 가치가 높습니다.

---

### **1. 총 경관 면적 (LND_TA: Total Area)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 분석 대상이 되는 전체 경관의 면적(ha)을 의미합니다. 이 값이 크다는 것은 농업 활동이 이루어지거나 잠재적으로 가능한 공간의 총량이 넓다는 것을 의미하므로, 농업 자원의 총량을 나타내는 기본 지표입니다.
- **인용:** `FRAGSTATS Manual`

### **2. 클래스 총면적 (CLS_CA: Class Area)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 유형(예: 1등급 토양)에 해당하는 농지의 총면적(ha)입니다. 해당 유형의 농업 자원이 얼마나 풍부한지를 직접적으로 보여주는 지표입니다.
- **인용:** `LESA Handbook`, `우량농지 보전 정책`

### **3. 클래스 면적 비율 (CLS_PLAND: Percentage of Landscape)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 전체 경관 면적에서 특정 클래스가 차지하는 비율(%)입니다. 이 비율이 높을수록 해당 유형의 농지가 경관의 지배적인 요소임을 나타내며, 이는 집단화 및 규모화와 직결됩니다.
- **인용:** `FRAGSTATS Manual`

### **4. 최대 패치 비율 (CLS_LPI: Largest Patch Index)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 단일 농지(패치) 중 가장 큰 것이 전체 경관 면적의 몇 %를 차지하는지 나타냅니다. 이 값이 높을수록 파편화되지 않은 거대 우량 농지가 존재함을 의미하며, 이는 대규모 기계화 영농, 공동 영농, 용수 관리에 절대적으로 유리하여 생산 효율성과 직결됩니다.
- **인용:** `FRAGSTATS Intro`, `농지 보전 적지 분석`

### **5. 경관 최대 패치 비율 (LND_LPI: Largest Patch Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 전체에서 가장 큰 패치의 면적 비율입니다. Class 레벨과 마찬가지로, 경관의 파편화가 적고 지배적인 토지 이용 형태가 존재함을 시사합니다.
- **인용:** `FRAGSTATS Manual`

### **6. 평균 패치 면적 (CLS_AREA_MN: Area Mean)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 클래스에 속하는 개별 농지(패치)들의 평균 크기(ha)입니다. 클수록 평균 경작 규모가 크다는 의미로, 소규모 농지보다 생산성과 효율성이 높습니다.
- **인용:** `우량농지 보전 정책`

### **7. 패치 평균 면적 (PATCH_AREA_MN: Area Mean)**
- **레벨:** Patch
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 분석 대상이 되는 모든 개별 패치들의 평균 면적입니다. Class 레벨과 의미는 유사하나, 특정 유형에 국한되지 않고 평가 대상 전체의 평균적인 규모를 나타냅니다.
- **인용:** `우량농지 보전 정책`

### **8. 패치 면적 중앙값 (CLS_AREA_MD: Area Median)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 극단적으로 크거나 작은 값을 제외한 일반적인 농지의 크기를 나타냅니다. `AREA_MN`과 함께 농지의 전반적인 규모를 판단하는 데 사용됩니다.
- **인용:** `FRAGSTATS Manual`

### **9. 면적 가중 평균 면적 (CLS_AREA_AM: Area-Weighted Mean Area)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 큰 패치가 전체 통계에 더 큰 영향을 주도록 가중치를 부여한 평균 면적입니다. 소규모 자투리 농지보다 대규모 우량 농지의 특성을 더 강하게 반영하므로 농업적 가치 판단에 더 적합합니다.
- **인용:** `FRAGSTATS Manual`

### **10. 패치 면적 변이계수 (CLS_AREA_CV: Area Coefficient of Variation)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 농지(패치) 크기의 상대적인 변동성(%)을 나타냅니다. 이 값이 클수록 농지들의 크기가 불균일하고(들쭉날쭉하고), 작을수록 비슷한 크기의 농지들이 분포함을 의미합니다. 균일한 크기는 계획적인 영농 및 관리에 유리합니다.
- **인용:** `Trends in Metrics`

### **11. 총 패치 수 (LND_NP: Number of Patches)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** 경관 전체의 패치(토지 조각) 총 개수입니다. 많을수록 경관이 잘게 쪼개져 있음을 의미합니다.
- **인용:** `FRAGSTATS Intro`

### **12. 클래스 패치 수 (CLS_NP: Number of Patches)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 동일한 총면적이라도 패치 수가 많다는 것은 농지가 그만큼 잘게 쪼개져 있다는 의미(파편화)입니다. 파편화는 관리 비용을 높이고 기계화 효율을 떨어뜨리는 핵심적인 부정 요인입니다.
- **인용:** `농지 보전 적지 분석`

### **13. 패치 밀도 (LND_PD: Patch Density)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** 단위 면적(100ha)당 평균 패치 수. `NP`와 같이 파편화의 정도를 나타내는 대표적인 지표로, 밀도가 높을수록 파편화가 심각함을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **14. 클래스 패치 밀도 (CLS_PD: Patch Density)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 특정 클래스의 단위 면적 당 패치 수. 역시 해당 유형 농지의 파편화 정도를 나타냅니다.
- **인용:** `FRAGSTATS Manual`

## 3. 형태 및 가장자리 (Shape & Edge Metrics)

농지의 형태와 가장자리 길이는 농업 효율성과 직접적인 관련이 있습니다. **단순한 형태(정사각형, 직사각형)일수록, 그리고 면적 대비 가장자리 길이가 짧을수록** 농업적 가치가 높습니다.

---

### **15. 평균 형태 지수 (CLS_SHAPE_MN: Shape Index Mean)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 패치 둘레를 동일 면적의 정사각형 둘레로 나눈 값. 1에 가까울수록 정사각형에 가까운 단순한 형태이며, 값이 클수록 복잡하고 비효율적인 형태임을 의미합니다. 복잡한 형태는 농기계 회전 반경을 늘리고 버려지는 공간을 만듭니다.
- **인용:** `FRAGSTATS Manual`

### **16. 패치 평균 형태 지수 (PATCH_SHAPE_MN: Shape Index Mean)**
- **레벨:** Patch
- **방향성:** **부정 (Negative)**
- **상세 근거:** 개별 패치들의 평균적인 형태 복잡도입니다. Class 레벨과 동일한 이유로 1에 가까울수록 좋습니다.
- **인용:** `FRAGSTATS Manual`

### **17. 면적 가중 평균 형태 지수 (CLS_SHAPE_AM: Area-Weighted Mean Shape Index)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 대규모 패치의 형태 복잡성에 더 큰 가중치를 부여한 값입니다. 대규모 농지의 형태가 단순할수록 농업적 가치가 높으므로, 이 값이 낮을수록 긍정적입니다.
- **인용:** `FRAGSTATS Manual`

### **18. 평균 프랙탈 차원 (CLS_FRAC_MN: Fractal Dimension Index Mean)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 둘레-면적 관계를 이용하여 형태 복잡성을 측정합니다. 1에 가까울수록 단순하며(선, 정사각형), 2에 가까울수록 복잡한 해안선 같은 형태를 의미합니다. 역시 1에 가까울수록 농업에 유리합니다.
- **인용:** `FRAGSTATS Intro`

### **19. 패치 평균 프랙탈 차원 (PATCH_FRAC_MN: Fractal Dimension Index Mean)**
- **레벨:** Patch
- **방향성:** **부정 (Negative)**
- **상세 근거:** 개별 패치들의 평균적인 프랙탈 차원. Class 레벨과 동일한 이유로 1에 가까울수록 좋습니다.
- **인용:** `FRAGSTATS Intro`

### **20. 면적 가중 평균 프랙탈 차원 (CLS_FRAC_AM: Area-Weighted Mean Fractal Dimension Index)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 대규모 패치의 프랙탈 차원에 가중치를 부여한 값입니다. 대규모 농지의 형태가 복잡할수록 비효율적이므로 낮을수록 좋습니다.
- **인용:** `FRAGSTATS Manual`

### **21. 평균 둘레 대 면적 비율 (CLS_PARA_MN: Perimeter-Area Ratio Mean)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 면적 대비 둘레의 길이. 클수록 동일 면적에 비해 가장자리가 길다는 의미로, 외부 환경(병충해, 잡초 등)에 대한 노출이 많고 실질 경작 면적이 비효율적임을 뜻합니다.
- **인용:** `FRAGSTATS Manual`

### **22. 패치 평균 둘레 길이 (PATCH_PERIM_MN: Perimeter Mean)**
- **레벨:** Patch
- **방향성:** **부정 (Negative)**
- **상세 근거:** 개별 패치의 평균 둘레 길이. 면적(`PATCH_AREA_MN`)과 함께 고려해야 하지만, 일반적으로 면적에 비해 둘레가 길면 비효율적인 형태로 간주할 수 있습니다.
- **인용:** `FRAGSTATS Manual`

### **23. 총 가장자리 길이 (LND_TE: Total Edge)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** 경관 내 모든 패치들의 가장자리 길이 총합입니다. 이 값이 크다는 것은 경관이 그만큼 파편화되어 있고, 패치 간 경계가 많음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **24. 클래스 총 가장자리 길이 (CLS_TE: Total Edge)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 특정 클래스에 속한 모든 패치들의 가장자리 길이 총합. 역시 파편화 및 형태 복잡성과 관련이 있습니다.
- **인용:** `FRAGSTATS Manual`

### **25. 가장자리 밀도 (LND_ED: Edge Density)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** 단위 면적(100ha)당 가장자리의 길이. 높을수록 경관이 복잡하고 파편화되었음을 나타내는 대표적인 지표입니다.
- **인용:** `FRAGSTATS Manual`

### **26. 클래스 가장자리 밀도 (CLS_ED: Edge Density)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 특정 클래스의 가장자리 밀도. 해당 유형 농지의 형태 복잡성 및 파편화 정도를 나타냅니다.
- **인용:** `FRAGSTATS Manual`

## 4. 핵심지역 (Core Area Metrics)

핵심지역은 패치의 가장자리(edge)로부터 일정 거리 안쪽의 영역으로, 외부 환경의 영향을 받지 않는 **안정적인 생산 공간**을 의미합니다. 핵심지역이 넓을수록 우량 농지로 평가됩니다.

---

### **27. 총 핵심지역 면적 (LND_TCA: Total Core Area)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 전체의 총 핵심지역 면적. 경관의 전반적인 생산 안정성을 나타냅니다.
- **인용:** `FRAGSTATS Manual`

### **28. 클래스 총 핵심지역 면적 (CLS_TCA: Total Core Area)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 해당 유형 농지의 실질적인 총 생산 공간의 크기(ha)입니다. 클수록 안정적인 영농이 가능한 면적이 넓음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **29. 핵심지역 면적 비율 (CLS_CPLAND: Core Area Percentage of Landscape)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 전체 경관 면적 중 특정 클래스의 핵심지역이 차지하는 비율(%)입니다. 경관 전체에서 해당 유형 농지가 차지하는 실질적 알짜 면적의 비중을 나타냅니다.
- **인용:** `FRAGSTATS Manual`

### **30. 평균 핵심지역 면적 (CLS_CORE_MN: Core Area Mean)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 개별 농지(패치)가 평균적으로 보유한 핵심지역의 크기(ha)입니다. 클수록 개별 농지들의 질이 높다고 할 수 있습니다.
- **인용:** `FRAGSTATS Intro`

### **31. 패치 평균 핵심지역 면적 (PATCH_CORE_MN: Core Area Mean)**
- **레벨:** Patch
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 개별 패치들의 평균 핵심지역 면적. Class 레벨과 동일한 이유로 클수록 좋습니다.

- **레벨:** Class

- **상세 근거:** 각 패치의 면적 중 핵심지역이 차지하는 비율(%)의 평균. 패치의 형태와 크기를 종합적으로 고려한 효율성 지표로, 높을수록 좋습니다.
- **인용:** `FRAGSTATS Manual`

### **33. 패치 평균 핵심지역 지수 (PATCH_CAI_MN: Core Area Index Mean)**
- **레벨:** Patch
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 개별 패치들의 핵심지역 비율 평균. Class 레벨과 동일한 이유로 높을수록 좋습니다.
- **인용:** `FRAGSTATS Manual`

### **34. 분리된 핵심지역 수 (CLS_NDCA: Number of Disjunct Core Areas)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 핵심지역이 몇 개의 조각으로 나뉘어 있는지를 나타냅니다. 많을수록 핵심지역마저 파편화되어 있다는 의미이므로 부정적입니다.
- **인용:** `FRAGSTATS Manual`

### **35. 분리된 핵심지역 밀도 (CLS_DCAD: Disjunct Core Area Density)**
- **레벨:** Class
- **방향성:** **부정 (Negative)**
- **상세 근거:** 단위 면적 당 분리된 핵심지역의 수. `NDCA`와 마찬가지로 핵심지역의 파편화 정도를 나타냅니다.
- **인용:** `FRAGSTATS Manual`

### **36. 패치 평균 분리된 핵심지역 수 (PATCH_NCORE_MN: Number of Core Areas Mean)**
- **레벨:** Patch
- **방향성:** **부정 (Negative)**
- **상세 근거:** 개별 패치 당 평균적으로 몇 개의 분리된 핵심지역을 갖는지를 나타냅니다. 높을수록 패치 내부가 비농업적 요소(도로, 수로 등)에 의해 단절되어 있음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

## 5. 집단 및 연결성 (Aggregation & Connectivity Metrics)

농지들이 서로 얼마나 모여 있고(집단화), 물리적으로 연결되어 있는지를 측정합니다. **농지는 집단화되고 서로 연결될수록** 공동 농작업, 용수 관리, 병충해 방제 등에 유리하여 가치가 높습니다.

---

### **37. 전염도 (LND_CONTAG: Contagion)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 내 패치들이 서로 인접한 정도를 나타냅니다. 높을수록 유사한 패치끼리 뭉쳐있어 파편화가 적고 집단화되어 있음을 의미합니다. 농지 관리에 매우 중요한 긍정적 지표입니다.
- **인용:** `농지 보전 적지 분석`, `FRAGSTATS Intro`

### **38. 집괴도 (CLS_CLUMPY: Clumpiness Index)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 유형의 패치(예: 1등급 농지)가 무작위 분포에 비해 얼마나 응집해 있는지를 나타냅니다. 1에 가까울수록 최대로 응집, -1에 가까울수록 최대로 분산되어 있음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **39. 집적 지수 (LND_AI: Aggregation Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 전체에서 동일 유형의 패치들이 얼마나 집적되어 있는지를 나타냅니다. 높을수록(100에 가까울수록) 집단화되어 있습니다.
- **인용:** `Trends in Metrics`

### **40. 클래스 집적 지수 (CLS_AI: Aggregation Index)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 클래스 패치들의 응집 정도를 측정합니다. 높을수록 집단화되어 있습니다.
- **인용:** `Trends in Metrics`

### **41. 인접 비율 (CLS_PLADJ: Percentage of Like Adjacencies)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 클래스의 총 가장자리 중, 같은 클래스와 맞닿은 가장자리의 비율입니다. 높을수록 동종의 농지끼리 모여있어 집단화 수준이 높음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **42. 응집도 (LND_COHESION: Patch Cohesion Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관의 물리적 연결성을 나타냅니다. 높을수록 농지 간 이동이 용이하고 단절되지 않았음을 의미하여 긍정적입니다.
- **인용:** `FRAGSTATS Manual`

### **43. 클래스 응집도 (CLS_COHESION: Patch Cohesion Index)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 클래스의 물리적 연결성. 높을수록 해당 유형의 농지들이 서로 연결되어 있음을 의미합니다.
- **인용:** `FRAGSTATS Manual`

### **44. 분할 지수 (LND_DIVISION: Division Index)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** 경관이 얼마나 분할되었는지를 나타내는 지표. 1에 가까울수록 완전히 분할(파편화)된 상태를 의미합니다.
- **인용:** `FRAGSTATS Intro`

### **45. 분리 지수 (LND_SPLIT: Splitting Index)**
- **레벨:** Land
- **방향성:** **부정 (Negative)**
- **상세 근거:** `DIVISION`과 유사하게 경관의 파편화 정도를 측정합니다. 총면적을 패치 수와 면적 제곱의 합으로 나누어 계산하며, 높을수록 파편화가 심합니다.
- **인용:** `FRAGSTATS Manual`

### **46. 유효 망 크기 (LND_MESH: Effective Mesh Size)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**

- **인용:** `FRAGSTATS Manual`

## 6. 다양성 및 복합성 (Diversity & Complexity Metrics)

경관 내에 얼마나 다양한 유형의 패치들이, 얼마나 균일하게 분포하는지를 나타냅니다. 농업 경관에서는 주변에 다양한 토지피복(산림, 초지 등)이 존재할 경우 생태적 완충 기능이나 생물 다양성 측면에서 긍정적으로 해석될 수 있습니다.

---

### **47. 샤논 다양성 지수 (LND_SHDI: Shannon's Diversity Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 패치 유형의 수와 각 유형의 면적 비율을 함께 고려하여 경관의 다양성을 측정합니다. 0 이상이며, 상한선은 없습니다.
- **인용:** `FRAGSTATS를 활용한 경관다양성의 시공간적 분석`

### **48. 심슨 다양성 지수 (LND_SIDI: Simpson's Diversity Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** `SHDI`와 유사하게 경관의 다양성을 측정하며, 0과 1 사이의 값을 가집니다.
- **인용:** `FRAGSTATS Manual`

### **49. 수정된 심슨 다양성 지수 (LND_MSIDI: Modified Simpson's Diversity Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** `SIDI`를 변형한 지수로, 해석은 동일합니다.
- **인용:** `FRAGSTATS Manual`

### **50. 샤논 균등도 지수 (LND_SHEI: Shannon's Evenness Index)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 내 다양한 패치 유형이 얼마나 균일한 면적으로 분포하는지를 나타냅니다. 특정 유형이 압도적이지 않고 여러 유형이 고르게 존재할수록 1에 가까워집니다.
- **인용:** `FRAGSTATS Manual`

### **51. 패치 풍부도 (LND_PR: Patch Richness)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 경관 내에 존재하는 패치 유형의 총 수.
- **인용:** `FRAGSTATS Manual`

### **52. 패치 풍부도 밀도 (LND_PRD: Patch Richness Density)**
- **레벨:** Land
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 단위 면적 당 패치 유형의 수. 다양한 토지 이용 유형의 존재는 생태계 서비스(수분 조절, 오염 정화 등) 측면에서 긍정적일 수 있습니다.
- **인용:** `FRAGSTATS Manual`

### **53. 이웃지수 (CLS_IJI: Interspersion and Juxtaposition Index)**
- **레벨:** Class
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 특정 클래스 주변에 얼마나 다양한 종류의 다른 패치들이 인접해 있는지를 나타냅니다. 높을수록 주변 패치 유형이 다양함을 의미합니다.
- **인용:** `FRAGSTATS Manual`

- **레벨:** Patch
- **방향성:** **긍정 (Positive)**
- **상세 근거:** 개별 패치들의 평균적인 공간 확산 범위. Class 레벨과 동일한 이유로 클수록 좋습니다.
- **인용:** `FRAGSTATS Manual`

### **56. 회전반경 중앙값 (CLS_GYRATE_MD: Gyrate Index Median)**

- **인용:** `FRAGSTATS Manual`

### **57. 면적 가중 평균 회전반경 (CLS_GYRATE_AM: Area-Weighted Mean Gyrate Index)**

- **인용:** `FRAGSTATS Manual`
