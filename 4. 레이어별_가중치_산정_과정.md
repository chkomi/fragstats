# 4. 레이어별 가중치 산정 과정 및 결과

이전 분석(`3. 최종_분석_보고서_상세.md`)에서 우리는 8개 평가 대상(지역-카테고리 조합)에 대한 '종합 농업가치 점수'를 도출했습니다. 이 점수는 각 대상의 농업적 보전 가치를 정량적으로 나타냅니다.

사용자님의 요청에 따라, 이 결과를 바탕으로 4개 레이어(`infra`, `nongeup`, `toyang`, `pibok`) 각각의 상대적 중요도, 즉 **'레이어 가중치'**를 산정하는 과정과 그 결과를 제시합니다. 이 가중치는 향후 GIS 가중 중첩 분석에서 각 레이어의 영향력을 결정하는 데 사용될 수 있습니다.

---

### 1. 분석 방법론

레이어 가중치를 산정하는 방법론은 다음과 같습니다.

1.  **레이어별 평균 점수 계산:**
    이전 분석에서 도출된 8개 평가 대상의 최종 점수를 사용합니다. 각 레이어(`infra`, `nongeup` 등)에 대해 나주와 화순 지역의 점수 평균을 계산하여, 해당 레이어의 전반적인 '농업적 가치'를 대표하는 단일 점수를 구합니다.
    *   `레이어 평균 점수 = (나주 지역 점수 + 화순 지역 점수) / 2`

2.  **평균 점수 정규화 (가중치 산정):**
    계산된 4개 레이어의 평균 점수들을 모두 더한 총합을 구합니다. 그 후, 각 레이어의 평균 점수를 총합으로 나누어 표준화(normalize)합니다. 이 표준화된 값이 바로 각 레이어의 최종 가중치가 됩니다. (가중치의 총합은 1)
    *   `레이어 가중치 = (해당 레이어의 평균 점수) / (모든 레이어 평균 점수의 총합)`

이 방법은 최종 분석 결과를 직접 활용하므로, 58개 지표의 복합적인 특성이 모두 반영된 상태에서 각 레이어의 상대적 중요도를 데이터 기반으로 도출하는 합리적인 방식입니다.

### 2. 계산 과정

#### 2.1. 원본 데이터: 평가 대상별 종합 점수

| 평가 대상 ID | 종합 점수 |
| :--- | :---: |
| `naju_nongeup` | 74.08 |
| `hwasun_nongeup` | 62.41 |
| `hwasun_toyang` | 59.32 |
| `naju_toyang` | 56.53 |
| `hwasun_infra` | 44.64 |
| `naju_infra` | 38.81 |
| `hwasun_pibok` | 28.03 |
| `naju_pibok` | 23.54 |

#### 2.2. 레이어별 평균 점수 계산

*   **Nongeup (농업용도) 평균 점수:**
    `(74.08 + 62.41) / 2 = 68.245`
*   **Toyang (토양) 평균 점수:**
    `(59.32 + 56.53) / 2 = 57.925`
*   **Infra (기반시설) 평균 점수:**
    `(44.64 + 38.81) / 2 = 41.725`
*   **Pibok (토지피복) 평균 점수:**
    `(28.03 + 23.54) / 2 = 25.785`

#### 2.3. 레이어 가중치 산정 (표준화)

*   **평균 점수 총합:**
    `68.245 + 57.925 + 41.725 + 25.785 = 193.68`

*   **최종 레이어별 가중치:**
    *   **W' nongeup** = 68.245 / 193.68 = **0.3524**
    *   **W' toyang** = 57.925 / 193.68 = **0.2991**
    *   **W' infra** = 41.725 / 193.68 = **0.2154**
    *   **W' pibok** = 25.785 / 193.68 = **0.1331**

### 4. 해석 및 활용 방안

분석 결과, **농업용도(Nongeup)** 의 가중치가 약 35.2%로 가장 높게 나타났습니다. 이는 농지의 등급과 같은 본질적인 농업 적합성이 전체 농업적 가치를 결정하는 데 가장 중요한 요인임을 의미합니다. 그 뒤를 이어 **토양(Toyang)** 이 약 29.9%로 높은 가중치를 차지했으며, **기반시설(Infra)** 과 **토지피복(Pibok)** 이 각각 21.5%, 13.3%로 뒤를 이었습니다.

이 가중치 값은 `4. 향후_분석_방향_및_정책_수립_방안.md`에서 제안한 GIS 가중 중첩 분석의 **핵심 파라미터**로 직접 사용할 수 있습니다.

`최종 점수 = (Infra 지도 × 0.215) + (Nongeup 지도 × 0.352) + (Toyang 지도 × 0.299) + (Pibok 지도 × 0.133)`

위와 같은 방식으로 '종합 농업가치 지도'를 생성함으로써, 각 레이어의 데이터 기반 중요도를 반영한 한층 더 정교하고 객관적인 공간 분석이 가능해집니다.
