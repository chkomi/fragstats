# 3. 최종 분석 보고서 (상세)

**분석 일시:** 2025-11-08 08:31:24

## 1. 분석 개요

본 문서는 나주 및 화순 지역의 농업진흥지역 내 농지 적합 유형(기반시설 수혜, 1등급 토양/용도/피복)에 대한 농업적 가치를 평가한 상세 분석 보고서입니다.

FRAGSTATS의 Class, Land, Patch 세 가지 레벨에서 도출된 58개의 모든 경관지표를 통합하고, 객관적인 가중치 산정을 위해 엔트로피 가중치법(Entropy Weight Method)을 적용하여 종합 점수를 산출했습니다. 이 문서는 논문 등에 인용할 수 있도록 전체 계산 과정과 모든 지표의 가중치를 투명하게 공개합니다.

---

## 2. 분석 방법론 및 계산 과정

### 2.1. 데이터 통합 및 행렬 구축

1.  **데이터 소스:** 나주/화순 2개 지역의 4개 카테고리(기반시설, 토양, 농업용도, 토지피복)에 해당하는 FRAGSTATS 분석 결과 파일 (`*_class.txt`, `*_land.txt`, `*_patch.txt`)
2.  **평가 대상:** 농지 적합 유형인 `giban_benefited` 또는 `cls_1`에 해당하는 총 8개 평가 대상(e.g., `hwasun_infra`, `naju_toyang` 등)
3.  **지표 통합:** 3개 레벨의 지표를 다음과 같이 통합하여 총 58개의 지표를 사용했습니다.
    *   **Class Level (31개):** `CLS_` 접두사 부여 (예: `CLS_LPI`)
    *   **Land Level (19개):** `LND_` 접두사 부여 (예: `LND_CONTAG`)
    *   **Patch Level (8개):** 각 평가 대상에 해당하는 모든 패치의 지표별 **평균값**을 계산하여 사용. `PATCH_*_MN` 접두사 부여 (예: `PATCH_AREA_MN`)
4.  **데이터 행렬 구축:** 위 과정을 통해, 행은 8개 평가 대상, 열은 58개 통합 지표로 구성된 **8x58 데이터 행렬**을 구축했습니다.

### 2.2. 엔트로피 가중치법 적용

1.  **지표 방향성 정의:** 58개 모든 지표에 대해, 값이 클수록 긍정적인지(`positive`), 작을수록 긍정적인지(`negative`)를 경관생태학 이론에 기반하여 정의했습니다. (예: `CLS_AREA_MN`은 positive, `CLS_NP`는 negative)

2.  **데이터 정규화 (Normalization):** 각 지표(열)별로 전체 평가 대상(8개)의 값을 0과 1 사이의 값으로 변환하여 단위와 범위를 통일했습니다.
    *   `positive` 지표: `정규화 값 = (원본 값 - 최소값) / (최대값 - 최소값)`
    *   `negative` 지표: `정규화 값 = (최대값 - 원본 값) / (최대값 - 최소값)`

3.  **엔트로피 및 가중치 계산:**
    *   **비중(P):** 각 지표(열) 내에서 개별 정규화 값이 차지하는 비율을 계산합니다.
    *   **엔트로피(E):** `E = -k * Σ(P * ln(P))` 공식을 통해 각 지표의 정보량을 산출합니다. 값의 차이가 적어 변별력이 낮을수록 엔트로피는 1에 가깝습니다.
    *   **분산도(D):** `D = 1 - E` 로, 정보의 유효성을 나타냅니다.
    *   **최종 가중치(W):** `W = D / ΣD` 로, 각 지표의 분산도를 전체 분산도 합으로 나누어 최종 가중치를 도출합니다.

4.  **종합 점수 산출:** 각 평가 대상에 대해 `Σ (지표별 정규화 값 × 지표별 가중치)` 공식을 적용하여 100점 만점의 최종 종합 점수를 계산했습니다.

## 4. 결론 및 해석

분석 결과, **농업용도(`nongeup`)** 와 **토양(`toyang`)** 카테고리가 다른 카테고리에 비해 전반적으로 높은 점수를 받았습니다. 이는 두 지역 모두에서 농지의 본질적 기능인 토양의 질과 농업 적합성이 농업적 가치를 결정하는 데 가장 중요한 요소임을 시사합니다.

반면, **토지피복(`pibok`)** 점수가 가장 낮게 나타났는데, 이는 현재 토지가 임야나 초지와 같은 비농업적 요소와 많이 혼재되어 있어 파편화가 심하고 실질적인 농업 활동 공간(핵심지역)이 부족함을 의미합니다. **기반시설(`infra`)**은 중간 정도의 점수를 보여, 기반시설의 유무가 농업적 가치에 영향을 주지만, 현재 분석 대상지에서는 농지 자체의 특성만큼 결정적이지는 않음을 보여줍니다.

지역별로는 나주와 화순이 각 카테고리 내에서 유사한 순위를 보였으나, 농업용도에서는 나주가, 토양에서는 화순이 근소하게 높은 점수를 받았습니다.

결론적으로, 본 분석은 농지의 가치를 평가할 때 단순히 특정 지표나 레벨에 국한되지 않고, **면적, 형태, 핵심지역, 연결성 등 다차원적인 경관 특성을 종합적으로 고려**해야 함을 정량적으로 보여줍니다. 특히 `CLS_AREA_AM`(대규모 패치의 존재 여부)과 `CLS_LPI`(가장 큰 단일 패치의 중요성)가 핵심적인 영향 요인으로 도출되었습니다.
